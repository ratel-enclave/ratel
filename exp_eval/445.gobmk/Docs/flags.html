445.gobmk Flags

These are flags that appear in the benchmark sources, categorized as to whether they are portability candidates, mandatory, or forbidden.

Portability Flags

  • -DHAVE_SYS_TIME_H
  • -DHAVE_VISUAL_C
  • -DVMS
  • -D__GLIBC__
  • -D__GNUC__
  • -D__GNU_LIBRARY__
  • -D__MINGW32__
  • -D__STDC__
    • -DHAVE_SYS_TIME_H
    • (?:^|(?<=\s))[-/]DHAVE_SYS_TIME_H(?:=\S*)?(?=\s|$)
    • Controls selection of sys/time.h or time.h .
    • -DHAVE_VISUAL_C
    • (?:^|(?<=\s))[-/]DHAVE_VISUAL_C(?:=\S*)?(?=\s|$)

    • Include Visual C header files.

    • -DVMS
    • (?:^|(?<=\s))[-/]DVMS(?:=\S*)?(?=\s|$)

    • Include correct libraries on VMS.

    • -D__GLIBC__
    • (?:^|(?<=\s))[-/]D__GLIBC__(?:=\S*)?(?=\s|$)

    • Is set to the major version of GNU C libraries.

    • -D__GNUC__
    • (?:^|(?<=\s))[-/]D__GNUC__(?:=\S*)?(?=\s|$)

    • Is set to the major version of gcc.

    • -D__GNU_LIBRARY__
    • (?:^|(?<=\s))[-/]D__GNU_LIBRARY__(?:=\S*)?(?=\s|$)

    • Asserts presence of GNU C libraries.

    • -D__MINGW32__
    • (?:^|(?<=\s))[-/]D__MINGW32__(?:=\S*)?(?=\s|$)

    • Is set to major version of MinGW.

    • -D__STDC__
    • (?:^|(?<=\s))[-/]D__STDC__(?:=\S*)?(?=\s|$)

    • Assert ANSI C standard is used.

Compiler Flags

  • -Dstrlen
    • -Dstrlen
    • (?:^|(?<=\s))[-/]Dstrlen(?:=\S*)?(?=\s|$)

    • Assert strlen function is already defined.

Mandatory Flags

  • -I., -I.., -I./include, -I../include
  • -DHAVE_CONFIG_H
  • -DSPEC_CPU_WINDOWS
    • -I., -I.., -I./include, -I../include
    • -I\.\.?(?:/include)?(?:\s+|$)

    • These flags are set for all builds in Spec/object.pm.

    • -DHAVE_CONFIG_H
    • (?:^|(?<=\s))[-/]DHAVE_CONFIG_H(?:=\S*)?(?=\s|$)

    • Use config.h supplied in src.

    • -DSPEC_CPU_WINDOWS
    • (?:^|(?<=\s))[-/]DSPEC_CPU_WINDOWS(?:=\S*)?(?=\s|$)
    • Define if system is running Windows. This will prevent the inclusion of unistd.h. It will also prevent the inclusion of both sys/time.h and time.h.

    • -DCHECK_HASHING

    • (?:^|(?<=\s))[-/]DELIDE_CODE(?:=\S*)?(?=\s|$)

    • Assert ELIDE code support

    • -DFULL_POSITION_IN_HASH
    • (?:^|(?<=\s))[-/]DFULL_POSITION_IN_HASH(?:=\S*)?(?=\s|$)

    • Implement adding full board position into hash table.

    • -DGG_GETOPT_H
    • (?:^|(?<=\s))[-/]DGG_GETOPT_H(?:=\S*)?(?=\s|$)

    • Prevents the header file gg-getopt.h from being loaded.

    • (?:^|(?<=\s))[-/]DGG_TURN_OFF_ASSERTS(?:=\S*)?(?=\s|$)

    • Turn off assert handling.

    • (?:^|(?<=\s))[-/]DGRID_OPT(?:=\S*)?(?=\s|$)

    • -DHAVE_LIBINTL_H
    • (?:^|(?<=\s))[-/]DHAVE_LIBINTL_H(?:=\S*)?(?=\s|$)

    • -DTEST
